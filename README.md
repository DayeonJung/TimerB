# TimerB
보드게임용 타이머 앱입니다.
1. 한 사람당 주어질 시간을 설정하고, 플레이할 인원수를 설정합니다. 최근 설정한 시간과 플레이어 수가 하단에 저장되므로, 그 중 하나를 골라도 타이머가 시작됩니다.
2. Game Start를 누르면 각 플레이어에게 주어진 시간만큼 물결이 줄어듭니다. 
3. 시간이 다 되면 다음 플레이어로 넘어갑니다.
